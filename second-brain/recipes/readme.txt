# Installation

1.
```
pip3 install paprika-recipes
nix-env -if https://github.com/DavHau/mach-nix/tarball/3.1.1 -A mach-nix
```
2. Ensure nix is installed
3. Look username and passWord up of paprika
4. Run `./rerun.sh --reset --upload`
5. This results into recipes being converted into markdown in ./markdown
